# E-Commerce Frontend

A responsive **E-Commerce web application** built using **React**, integrated with a Django REST API backend to deliver a complete online shopping experience.

---

 **Features**

- **User Authentication**
  - User registration and login
  - Secure access to user-specific features

- **Product Listing & Details**
  - Browse all available products
  - View detailed product information including price and description
  - Category-wise product browsing

- **Cart Management**
  - Add products to cart
  - Update product quantities
  - Remove products from cart

- **Wishlist**
  - Add products to wishlist
  - View and manage saved products

- **Orders**
  - Place orders from cart

- **Search & UI**
  - Search products by name
  - Responsive design for mobile and desktop devices

---

**Technologies Used**

- **Frontend:** React
- **Routing:** React Router
- **State Management:** React Hooks
- **Styling:** CSS
- **API Integration:** Fetch API
